Concepto de Angular
Angular es un framework de desarrollo para JavaScript creado por Google. La finalidad de este framework es facilitarnos el desarrollo de aplicaciones web SPA (Aplicaciones de una sola pagina).
Concepto de NodeJs
Es un entorno de tiempo de ejecucion de JavaScript, incluye todo lo que se necesita para construir aplicaciones basadas en JavaScript. Es el encargado de levantar nuestra aplicacion en Angular
Concepto de EcmaScript
Es el estandar que permite desde el anio 2015 a la actualidad, regir o especificar como debe ser interpretado y funcionar el lenguaje JavaScript.
Concepto de TypeScript
Es un superset o superconjunto de JavaScript, que esencialmente implementa tipado estatico y objetos basados en clases. Angular trabaja con TypeScript para la creacion de app. Al utilizar Typescript puede emplear tambien JavaScript nativo.
Comandos importantes
Accion Comando Descripcion
Nuevo Proyecto ng new nombre_proyecto Permite la creacion de un nuevo proyecto en Angular, la version del Angular depende de la version de nodejs. Aunque se puede especificar tambien la versión
Desplegar aplicacion ng serve -o Traspilar el codigo de TypeScript a JavaScript y si no hay errores abre una pestaña en el navegador por defecto
Crear componente ng g c carpeta/nombreComponente Crea un componente en la carpeta seleccionada y si la carpeta no esta creada, pues la crea.
Crear modulo ng g m carpeta/nombreModulo Crea un modulo en la carpeta seleccionada y si la carpeta no esta creada, pues la crea.
Crear un servicio ng g s carpeta/nombreServicio Crea un servicio en la carpeta seleccionada y si la carpeta no esta creada, pues la crea.
Crear una clase ng g class carpeta/nombreClase Crea una clase en la carpeta seleccionada
Instalar Bootstrap npm i bootstrap jquery popper.js Instalamos Bootstrap y sus dependencias
Instalar SweetAlert 2 npm --save install sweetalert2 Instalamos la libreria para mensajes o cuadros emergentes con un estilo moderno
Instalar Material ng add @angular/material Instalamos el framework Material para UI (Interfaz Grafica)
Instalar Font Awesome npm i @fortawesome/fontawesome-free --save Instalamos los iconos de Font Awesome
